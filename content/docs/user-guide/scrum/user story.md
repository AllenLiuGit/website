# 用户故事
  
  用户故事界面是用户故事的列表展现形式，在这里可以集中查看您创建的所有的用户故事，以及其下的任务缺陷等，方便快捷。
  
  - **菜单层次**：项目层
  - **菜单路径**：敏捷管理 > 用户故事
  - **默认角色**：看板项目所有者、看板项目成员
  
### 创建用户故事
  
  点击`创建故事`→ ![创建故事](/docs/user-guide/scrum/image/创建故事.png)，出现详情信息编辑的窗口，在窗口中进行编辑，编辑完成之后点击保存。

### 搜索用户故事
  
  点击`过滤`可以进行高级搜索
 
### 删除用户故事
    
  未进入冲刺的用户故事可以删除。

### 快捷创建任务
    
  可以快捷创建用户故事下面的任务。

### 快捷创建缺陷
    
  可以快捷创建用户故事下面的缺陷。
