# 节点监控

## 操作指南

左上角 `chluster` 菜单可切换集群， `node` 菜单可切换集群中的节点 。

## 监控指标

### Memory Used

显示已用内存

### Memory

节点总内存

### CPU Used

显示已用CPU（每个核心）

### CPU CORES

内存核心数

### Swap Used

显示已用内存交换空间

### Swap

内存交换区大小

### Disk on '/' Used

显示 '/' 路径下硬盘占用

### File System

文件系统大小

### nodexx cpu

节点cup使用折线图

### Pod Cpu

节点上运行的pod占用cpu折线图

### Pod Memory Percentage

节点上运行的pod使用的内存相对主机内存占比折线图

### Pod Memory

Pod内存使用量折线图

### Node Network

节点网络收发总折线图

### Pod Network Receive

节点中Pod网络接收使用量折线图

### Pod Network Trasmit

节点中Pod网络发送使用量折线图
