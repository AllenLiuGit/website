# 应用版本
 
 应用版本是应用的版本记录，每提交一次修改后的代码，对应生成一个新的版本。

  - **菜单层次**：项目层
  - **菜单路径**：持续交付 > 应用版本
  - **默认角色**：项目所有者、源代码管理员、项目成员

### 查看应用版本信息

 1. 进入持续交付后，点击 `应用版本` 页签。

 1. 查看应用版本信息。

列表字段

 - 应用版本：迭代升级中对应的不同版本。

 - 应用编码：创建应用的自定义编码；

 - 应用名称：创建应用的自定义名称；

 - 创建时间：应用创建时的时间；
