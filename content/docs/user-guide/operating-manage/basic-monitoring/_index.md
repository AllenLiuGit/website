# 服务指标

<h2 id="1">介绍</h2>

使用prometheus进行指标采集，然后使用grafana进行数据图形化展示。

grafana是基于JS开发的，功能齐全的度量仪表盘，可以对接多数据源，实现图形化实时监控。

<h2 id="2">模块</h2>

 - [**节点监控**](../basic-monitoring/node) 显示 kubernetes 集群中的节点指标。

 - [**节点概述监控**](../basic-monitoring/allnode) 显示 kubernetes 集群中的节点指标。

<h2 id="2">使用简介</h2>
登录后点击看板切换菜单即可切换监控模块。


**菜单中**

`node` 为节点监控模块

`All Node` 为节点概述监控模块
