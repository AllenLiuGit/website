# 项目

项目管理，即对项目的信息、项目的使用的管理，但不管理项目的成员。

**组织层的功能：**

- [创建项目](#1)
- [其他操作](#2)


<h2 id="1">创建项目</h2>

**必填项：**

- 项目编码：
    - 具有唯一性
    - 不能包含大写字母、中文、特殊字符

- 项目名称：
    - 具有唯一性
    - 项目名称长度限定为4-30字符

![创建项目](../images/2-2.1_1.png)

<h2 id="2">其他操作</h2>

- **编辑：**项目代码一旦创建,不可编辑，只可编辑项目名称

- **搜索：**
    - 不选择属性，根据输入值进行全局搜索
    - 选择一个属性：项目ID（创建后自动生成）、项目编码、项目名称，输入相应的属性值进行搜索

- 启用/停用：项目停用后，项目信息仍在项目列表中，但是不可应用该项目或进入该项目进行操作

 > 注意：项目一旦创建，不可删除
