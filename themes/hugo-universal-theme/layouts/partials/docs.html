Choerodon 文档

Choerodon 企业数字化平台有两部分组成，其一是Choerodon DevOps 平台；其二是HAP Cloud 微服务应用框架。Choerodon 文档将详细介绍Choerodon DevOps平台的架构、安装、配置、部署，以及如何使用等，并且介绍HAP Cloud微服务应用框架的架构、组件、开发、部署等。

关于Choerodon的介绍，请参考 Choerodon概览。
